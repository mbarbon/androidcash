AndroidCash
===========

A simple expense tracker application for Android; it is meant to enter
expense entries that can be later imported in GnuCash and deleted from
the device.

Given the limited scope, it lacks many advanced features provided by
other stand-alone applications.

The source code for this program is available on GitHub
(http://github.com/mbarbon/androidcash) and is distributed under the
MIT license.
